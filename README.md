# nltk-backlog-processing

Processing a scrum backlog with natural language processing.

## How to setup

To setup, create a new virtual environment and add the required dependencies:

```
python -m venv env
source env/bin/activate

pip install nltk
```

## How to run

From the repository root, ensure the virtual environment is active and run:

```
python stemmed.py
```

This will process the sample tasks located in `py/sample.py` and print the result to the terminal.
