Sprint Backlog - Todo App (Sprint 5)


User Stories
- As a user, I want to create a new task so that I can track things I need to do
- As a user, I want to edit existing tasks so that I can update details
- As a user, I want to delete tasks so that I can remove completed or irrelevant items
- As a user, I want to mark tasks as complete so that I can track progress
- As a user, I want to set due dates so that I can manage deadlines
- As a user, I want to organize tasks into categories so that I can group related items
- As a user, I want to search for tasks so that I can quickly find specific items
- As a user, I want to filter tasks by status so that I can view completed or pending items
- As a user, I want to receive notifications for upcoming deadlines so that I don't miss tasks
- As a user, I want to sync my tasks across devices so that I can access them anywhere
